## How to use

### Create Box

If you use distrobox:

    distrobox create -i ghcr.io/grantmacken/tbx:v3.18.4 -n tbx
    distrobox enter tbx
    
If you use toolbx:

    toolbox create -i ghcr.io/ublue-os/boxkit -c boxkit
    toolbox enter boxkit
